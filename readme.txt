Wordpress Kullanıcılarına özel Menü sayfası veya sadece ziyaretçilere gösterebileceğiniz menü sayfaları.

== Description ==

Bu eklenti Wordpress kullanıcılarınıza veya ziyaretçilerinize menüde gösterebileceğiniz sayfaları ayarlar.
Sadece kullanıcılarım "Hesabım" sayfasını görecekse küçük bir metin ile bunu ayarlarsınız ve sadece kullanıcılarınız bu sayfayı görebilir.

== Installation ==

Aşağıdaki adımlara bakarak eklentiyi yükleyebilirsiniz.

1. Eklentiyi '/wp-content/plugins/' adlı adrese atmanız gerek, yada Wordpress paneli içinden eklentinin ismini yazarakta yükleyebilirsiniz..
2. Eklentiyi aktifleştirmek için 'Eklentiler' bölümünden gerekli eklentiyi aktifleştirebilirsiniz.


1. İlk Adımda "Görünüm > Menüler" geliyoruz ve "Ekran Tercihleri" Butonuna Tıklıyoruz. <img src="/assets/screenshot-1.png">
2. İkinci Adımda "Ekran Tercihleri > CSS Sınıfları" Etkinleştiriyoruz. <img src="/assets/screenshot-2.png">
3. Üçüncü Adımda Kullanıcının veya Ziyaretçinin Görmesini istediğimiz sayfaya gelip tıklıyoruz aşağı doğru açılıyor ve Css Sınıfları Bölümünden kim görecekse Kurulumlar bölümündeki kodu yapıştırıyoruz. <img src="/assets/screenshot-3.png">

Kullanım: 
1.Sadece Ziyaretçinin Görmesini İstediğimiz Css Sınıfı "visitor".
2.Sadece Kullanıcının Görmesini İstediğimiz Css Sınıfı "registered_user".

Resimli Kullanım Screenshots Bölümündedir.
